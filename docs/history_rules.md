# TradeLens History Rules

- `analysis_history/` contains one saved markdown file per analysis.
- `/tradelens history` lists today's records.
- `/tradelens history all` lists all records in reverse chronological order.
- `/tradelens history --name` shows one exact or fuzzy record.
- `/tradelens history feedback --name` shows one record and records feedback.
- History commands do not fetch market data.
- Saved analysis detail must preserve the standard disclaimer exactly once.
- Feedback choices are `accurate`, `not accurate`, and `no feedback`.
